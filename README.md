## 🌐 How to Run

### Locally (VS Code)
1. Open folder in VS Code
2. Install **Live Server** extension
3. Right-click `index.html` → **Open with Live Server**

### Directly in Chrome
Double-click `index.html` — opens directly in browser!

### GitHub Pages (Free Hosting)
1. Push to GitHub
2. Settings → Pages → Source: `main` branch
3. Your site is live at `https://yourusername.github.io/CyberShield-Sentinel/`
